# NEXAH Stability Engine — Visual Output Index

This document describes the visual outputs generated by the NEXAH Stability Engine.

All figures are produced by:

ENGINE/run_stability_engine.py

The images are stored in:

ENGINE/visuals/

Each visualization represents a structural aspect of the stability landscape.

---

# 01 — Stability Landscape

File  
01_landscape.png

Description  
The base stability landscape Z = f(x,y).

The surface represents the global stability structure of the system.

Bright regions correspond to high stability (or energy depending on interpretation).  
Dark regions correspond to lower stability.

Purpose  
Provides the global field on which all further analysis is performed.

---

# 02 — Gradient Field

File  
02_gradient_field.png

Description  
The gradient field ∇f(x,y) of the landscape.

Vectors show the direction of steepest ascent or descent.

Purpose  
Indicates the natural flow directions of the system and reveals local motion tendencies.
